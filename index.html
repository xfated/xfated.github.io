EQUIPMe

Gym Equipment Classifier

This humble project aims to help beginner gymmers identify the various machines found at the gym!
Equipme is a CNN classifier that uses transfer learning from the VGG16 model trained on imagenet dataset to regocgnize categories of gym equipment. It is currently able to detect 8 different categories of gym equipment using images sourced from the web. Though the validation accuracy reached 99%, the online images might not be representative of real-life images.
